Information

This is My To Do List App with items generated using angularjs, and displayed in a simple HTML list. We can add and delete items from the To Do List App.

To add an item to the list, just type the item's name in the input box provided and click the "Add Item" button.

If the item is already included in the list, it will not be added, because error handling technique will report that it is already in the list and should not be included.

To delete an item, just click on the little (x) next to it.
